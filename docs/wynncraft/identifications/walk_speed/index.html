Walk Speed

    Minimum -100%
    Maximum 400%
    The cap for Walk Speed can be bypassed by any item that provides the Speed effect, which includes:
        Trinkets that say they give Speed I,
        Windy Feet,
        Stormy Feet.
    The Walk Speed Cap is not bypassed by the following:
        +Freerunner Major Id.,
        Time Dilation.
    Needs to be tested:
        Mask of the Heretic.
    To calculate the speed at which a player will sprint with a given Walk Speed %, the following equation can be used:
        blocks/second = 5.612 * ((walkSpeed / 100) + 1)
